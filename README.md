# GitHub Pages — Portal del Códice Maestro (estático)

Este repositorio es **100% estático** (sin backend). GitHub Pages sirve el sitio desde `index.html` en la raíz.

## Publicar

1. GitHub → Settings → Pages
2. Source: Deploy from a branch
3. Branch: `main`
4. Folder: `/(root)`

## Rutas

- PDFs: `assets/deliverables/`
- Modelos 3D: `assets/models/`
- Imágenes: `assets/images/` y `assets/images/source/`

## Nota

Los PDFs fueron convertidos desde documentos Word (fuente) usando LibreOffice (headless). Los archivos Word no se versionan en este repositorio; solo se publican PDFs e imágenes.
